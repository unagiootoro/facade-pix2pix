# facade-pix2pix
Perform image conversion of facade dataset with Pix2pix and ruby-dnn.

# Try
### Preparing for required installation
・ruby-dnn  
・numo-linalg  
・rubyzip  

### Clone repository
```
$ git clone https://github.com/unagiootoro/facade-pix2pix
```

### Start image conversion
```
$ ruby imgen.rb
```
